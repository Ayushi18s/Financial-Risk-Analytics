# 💰 Financial Risk Analytics Dashboard | SQL + Python + Power BI

## 📌 Project Overview

This project focuses on analysing bank loan data to uncover lending trends, borrower behaviour, loan performance, and financial risk insights using SQL, Python, and Power BI.

The objective of this project is to transform raw financial data into meaningful business insights through KPI reporting, exploratory data analysis, and interactive business intelligence dashboards.

---

## 🎯 Business Problem

Financial institutions require continuous monitoring of loan applications, funded amounts, borrower repayment behaviour, and portfolio risk to make data-driven lending decisions.

This project helps analyse:

* Loan application trends
* Good vs Bad loans
* Borrower financial behaviour
* Regional lending patterns
* Loan repayment performance
* Financial risk indicators
